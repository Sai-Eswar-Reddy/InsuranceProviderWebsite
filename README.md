# 🛡️ Insurance Provider Website

## 📌 Overview

A responsive front-end web application designed to present various insurance services through a clean and user-friendly interface. The project focuses on UI/UX design, layout structuring, and responsive web development using Bootstrap.

---

## 🎯 Key Features

* Fully responsive design across mobile, tablet, and desktop devices
* Clean and modern UI built using Bootstrap components
* Multiple pages for different insurance categories (Life, Health, General, etc.)
* Structured navigation with intuitive user flow
* Informational pages such as claim process and agent details
* Consistent styling and layout across all pages

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Framework:** Bootstrap
* **Version Control:** Git & GitHub

---

## 🎨 UI/UX Highlights

* Responsive grid system using Bootstrap
* Reusable components (navbar, cards, forms, sections)
* Proper spacing, alignment, and typography for better readability
* Mobile-first design approach

---

## 📂 Project Structure

* `/assets` – Stylesheets and static resources
* `/forms` – Form-related pages
* HTML Pages:

  * `index.html` – Home page
  * `signin.html`, `signup.html` – UI forms
  * `claim-process.html` – Claim information
  * Additional pages for different insurance types

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in any web browser
